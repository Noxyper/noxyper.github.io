Projects About Me CV

Anything World - Unity Plugin

Developed the tool for Unity users to interface with the capabilities of Anything World, ranging from streaming in assets from the web to ML-powered automatic rigging.

Battlesweeper

Developed as part of Grads in Games 'Search for a Star' competition in 2022. A combination of Battleships and Minesweeper, the game showcases my proficiencies with local multiplayer capabilities and game AI.

University Game Jam

Annually, I've taken part in the University of Portsmouth Game Jam, a week-long game jam with a focus on healthy development cycles and experimentation. Taken part both in teams and individually.

Fearing Mirrors

A game story developed during the Game Writing course headed by Lorne Nudel. Developed script work, character biography, and lore item descriptions for a game about identity and dysmorphia.

Unity Experiments

A collection of the different coursework I developed throughout my BSc in Unity. Coursework touches upon gameplay programming fundamentals, linear algebra, tools development, and game AI.
